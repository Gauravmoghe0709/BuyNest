✅ What is Jest?

Jest is a JavaScript testing framework used mainly for testing JavaScript and Node.js applications.

✅ Why is Jest used?

To test functions and components

To check API responses

To catch bugs before deployment

Very fast and easy to use




✅ What is Redis?
 Redis is a fast, in-memory database used for caching, storing sessions, queues, rate limiting, and real-time applications.


 ✅ Why Redis is used?
✔ 1. Caching

To store frequently used data so your app becomes fast.

Example:
Save user profile in Redis → next time no need to query database.

✔ 2. Session Storage

Store user login sessions (better than storing in server memory).

✔ 3. Rate Limiting

Example:
Allow only 100 API calls per minute for a user.

✔ 4. Queues / Messaging

Used in background jobs (Bull, Agenda, etc.)

✔ 5. Real-time apps

Used in chat apps, notifications, leaderboards, stock data, etc.
